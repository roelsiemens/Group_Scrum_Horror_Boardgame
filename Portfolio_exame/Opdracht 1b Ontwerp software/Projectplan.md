## Definition of done

Een issue is klaar wanneer:

- De acceptatiecriteria van de issue zijn behaald.
- Er is een pull request gemaakt voor de issue.
- De pull request is nagekeken door een teamgenoot.
- De code is geaccepteerd.
- De pull request is naar de merge branch gemerged door degene die de pull request
gemaakt heeft
